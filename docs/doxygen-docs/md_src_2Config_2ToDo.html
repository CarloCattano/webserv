The important part is to set up Nginx to pass the correct information to FastCGI using the fastcgi_param directive. For example: location ~ .php$ { include /etc/nginx/fastcgi_params; fastcgi_pass 127.0.0.1:9000; fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name; fastcgi_param PATH_INFO $fastcgi_path_info; fastcgi_param PATH_TRANSLATED $document_root$fastcgi_path_info; fastcgi_split_path_info ^(.+.php)(.*)$; } This example shows how to pass a .php file to a PHP-CGI handler. The fastcgi_param directives pass additional information to the PHP interpreter.
