Detailed Description

This class is responsible for handling the server cluster. It contains the server map, client map, pipeFd_clientFd_map, cgi_response_map, epoll_fd and the methods to handle the server cluster.

Parameters
_server_mapmap of server objects where the servers instantiated are stored
_client_mapmap of clients with a file descriptor as the key and the client object as the value
_pipeFd_clientFd_mapmap of pipe file descriptors and client file descriptors
_cgi_response_mapmap of cgi responses with the client file descriptor as the key and the response as the value
_epoll_fdepoll file descriptor
